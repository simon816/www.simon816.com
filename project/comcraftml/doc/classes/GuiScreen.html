Add an action handler to the screen on selected screens.
Used as a GuiButton callback method.
Screens that fire this event can be found in the above list.

(
  • GuiName
)

Test whether this GuiScreen has the name GuiName
Use this to target your desired screen when Comcraft.displayScreen is fired. The following is a list of all names:

Properties

cc

Comcraft

The instance of Comcraft used by the screen.

elementsList

Array

The list of elements drawable on the screen.
Modify this array in initGui to change elements. After initGui, modifying this array will have NO effect.

Attributes

initGui

Function required

Set the initGui event.

Events

initGui

Fired when the GuiScreen is first initialised.
No payload is declared, but importantly the this pointer points to the GuiScreen being initialised.
