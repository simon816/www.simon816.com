importFile

(
  • pkgName
  • fileName
)
Boolean

Import a file into the global namespace.
Throws Error When the import failed

Parameters:

  • pkgName String

    the name of the package

  • fileName String

    the name of the 'class'

Returns:

Boolean:

Whether the file was imported or not

Example:

try {
     if (importFile("com.simon816", "common")) {
             console.info("Successfully imported com.simon816.common")
     } else {
             console.warn("com.simon816.common does not exist. Please install required mod.")
     }
} catch (err) {
     console.error("com.simon816.common is corrupt. Import failed", err)
}
